2.0 Compiling code for ubuntu on Linux
################################################################################

First time:

1. Modify code on Mac for Eclipse for red5
2. Copy $(ProjectPath)/target/red5-server-1.0.2-RC4-server.tar.gz to unbuntu
3. Unzip the folder to /usr/share/red5
4. Copy myRed5App under /user/share/red5/webapps

Later on:

1. Modify code on Mac for Eclipse for red5
2. Copy $(ProjectPath)/target/red5-server-1.0.2-RC4.jar to unbuntu 
3. On Unbuntu, $cp red5-server-1.0.2-RC4.jar /home/share/red5/red5-server.jar
4. restart the server
